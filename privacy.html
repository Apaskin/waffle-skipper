Woffle — Privacy Policy

Last updated: March 2026  |  Version 0.1.0

What this extension does

Woffle is a Chrome extension that uses AI to classify segments of YouTube video transcripts as "substance" or "woffle" (filler), then automatically skips the filler segments while you watch.

Data we collect

We collect nothing. Woffle has no servers, no accounts, no analytics, and no telemetry. All processing happens locally in your browser.

Data stored on your device

The extension stores the following data locally in your browser only:

  • Your Anthropic API key — saved to Chrome's sync storage (chrome.storage.sync) so it persists across Chrome profiles if you use Chrome Sync. It is never transmitted anywhere except to the Anthropic API endpoint when analysing a video.
  • Video analysis cache — results of transcript classifications are cached locally in chrome.storage.local to avoid re-processing videos you've already watched. Cache entries expire after 30 days and are automatically cleaned up.

Data sent to third parties

When you navigate to a YouTube video, the extension fetches the video's English transcript (captions) and sends it to the Anthropic API for classification. Specifically:

  • The transcript text (not your identity, not the video URL, not any personal information) is sent to https://api.anthropic.com/v1/messages using your own API key.
  • Anthropic's data usage policy applies to this request. See anthropic.com/privacy.
  • No other data is sent to Anthropic or any other party.

YouTube interactions

The extension fetches YouTube video transcripts using YouTube's publicly available caption API. It does not log in to YouTube on your behalf, does not interact with your YouTube account, and does not store or transmit watch history.

Permissions used

  • storage — to save your API key and cache analysis results locally.
  • youtube.com host permission — to fetch video transcripts.
  • api.anthropic.com host permission — to call the Claude AI API for classification.

Children's privacy

This extension is not directed at children under 13. We do not knowingly collect any personal information from anyone.

Changes to this policy

If we change what data is collected or how it is used, we will update this page and increment the version number above.
