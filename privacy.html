WOFFLE — PRIVACY POLICY

Last updated: March 2026  |  Version 0.5.0

What Woffle does

Woffle is a Chrome extension that detects filler content ("woffle") in YouTube videos and lets you skip it. It captures the video's transcript, sends it to the Anthropic Claude API using your own API key, and displays the results as a colour-coded timeline overlay.

We never see your data. Woffle has no servers, no accounts, and no telemetry. Everything goes directly from your browser to Anthropic — under your account, using your key, billed to you.

What Woffle stores locally

The following data is stored in your browser only — never transmitted to us:

  • Your Anthropic API key — saved to chrome.storage.sync so it persists across your Chrome profiles if you use Chrome Sync. It is only ever sent to https://api.anthropic.com when analysing a video.
  • Your Woffle Pro license key (if purchased) — saved to chrome.storage.sync. Never transmitted anywhere.
  • Cached video analyses — segment classifications are cached in chrome.storage.local to avoid re-processing videos you've already watched. Cache entries expire after 30 days.
  • Your settings — auto-skip toggle, intensity level (light/medium/heavy), timeline visibility preference. Stored in chrome.storage.sync.

What Woffle does NOT do

  • No personal data collection of any kind
  • No tracking of your viewing habits or watch history
  • No transcript storage — transcripts are sent to Anthropic and discarded
  • No analytics, telemetry, or crash reporting
  • No data sharing with any third party except as described below

Third-party services

Woffle makes API calls to one external service:

  • Anthropic (api.anthropic.com) — receives the video transcript text for AI classification. This call is made using your API key, under your Anthropic account. Anthropic's own privacy policy applies: anthropic.com/privacy. No identifying information (name, email, YouTube account) is included in the request — only the transcript text.

YouTube interactions

Woffle fetches video transcripts using YouTube's publicly available caption API. It does not log in to YouTube on your behalf, does not interact with your YouTube account, and does not store or transmit watch history.

Permissions used

  • storage — to save your API key, license key, settings, and cached analysis results locally in your browser.
  • youtube.com host permission — to read video transcripts from YouTube's caption API and inject the timeline overlay into the YouTube page.
  • api.anthropic.com host permission — to call the Claude AI API for transcript classification using your API key.

Children's privacy

Woffle is not directed at children under 13. We do not knowingly collect any personal information from anyone.

Changes to this policy

If we change what data is collected or how it is used, we will update this page and increment the version number above.
